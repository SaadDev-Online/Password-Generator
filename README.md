# Password‑Generator

A simple web application that generates random secure passwords for users.

## Features

- Generate a strong password with a single click  
- Options to include uppercase letters, lowercase letters, numbers, and symbols  
- Copy the generated password to clipboard for easy use  
- Responsive design suitable for mobile and desktop  
- Plain HTML, CSS and JavaScript – easy to extend

Usage
Click the “Generate Password” button to produce a new password.

Optionally choose which character types to include (uppercase, lowercase, numbers, symbols).

Use the “Copy” button (if present) to copy the password to your clipboard.

Customize length and other settings as needed in script.js.
